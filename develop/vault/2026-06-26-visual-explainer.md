# visual-explainer

An agent skill that turns complex terminal output into styled HTML pages or slide decks for diagrams, diff reviews, plan audits, data tables, and project recaps.

**Stars:** 9,265 | **Forks:** 627 | **Language:** HTML | **License:** MIT

**Supported harnesses:** Claude Code (marketplace plugin), Pi, Codex CLI, OpenCode/opencode, Cursor, OpenClaw

**Commands:**
- `/generate-web-diagram` — HTML diagram for any topic
- `/generate-visual-plan` — visual implementation plan
- `/generate-slides` — magazine-quality slide deck
- `/diff-review` — visual diff review with architecture comparison
- `/plan-review` — compare plan against codebase with risk assessment
- `/project-recap` — mental model snapshot for context-switching
- `/fact-check` — verify accuracy of document against actual code

Uses Mermaid for flowcharts, CSS Grid for architecture, HTML tables for data, Chart.js for dashboards.
