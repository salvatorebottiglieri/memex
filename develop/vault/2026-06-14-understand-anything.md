# Understand Anything

An open-source project by Egonex that turns any codebase, knowledge base, or docs into an interactive knowledge graph you can explore, search, and ask questions about. Works with Claude Code, Codex, Cursor, Copilot, Gemini CLI, and more.

**Stars:** 74,859 | **Forks:** 6,226 | **Language:** TypeScript | **License:** MIT

**Key features:**
- Structural graph: navigate codebase as interactive knowledge graph — every file, function, class as a clickable node
- Business logic view: map code to real business processes (domains, flows, steps)
- Knowledge base analysis: parse Karpathy-pattern wikis into force-directed graphs with community clustering
- Guided tours: auto-generated walkthroughs ordered by dependency
- Fuzzy & semantic search
- Diff impact analysis
- Persona-adaptive UI
- Layer visualization by architectural layer
- 12 programming patterns explained in context

**Quick start:**
```
/plugin marketplace add Egonex-AI/Understand-Anything
/plugin install understand-anything
/understand
/understand-dashboard
```
